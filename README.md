# 🚀 My Personal Portfolio

Welcome to the repository for my personal portfolio! This site highlights my projects, technical skills, and experience as I complete my B.Sc. in Computer Science.

**🌐 Live Demo:** [sd-1604-portfolio.netlify.app](https://sd-1604-portfolio.netlify.app/)

---

## 🛠️ Tech Stack

- **Frontend:** HTML5, CSS3, JavaScript
- **Deployment:** Netlify
- **Version Control:** Git & GitHub

## 🌟 Key Features

- **Responsive Design:** Optimized for mobile, tablet, and desktop views.
- **Project Showcase:** A curated list of my technical projects with links to their respective repositories.
- **Contact Integration:** A functional contact section for professional inquiries.

## 📈 Local Setup

To run this project locally:

1. Clone the repo: `git clone https://github.com/SD1604/Portfolio-2026.git`
2. Open `index.html` in your browser.
